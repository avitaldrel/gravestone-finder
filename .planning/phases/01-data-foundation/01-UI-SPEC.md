## Spacing Scale

Declared values (multiples of 4, using Tailwind CSS 4 utility classes):

| Token | Value | Tailwind Class | Usage |
|-------|-------|----------------|-------|
| xs | 4px | gap-1, p-1 | Icon-to-label gaps, inline padding |
| sm | 8px | gap-2, p-2 | Compact element spacing, badge padding |
| md | 16px | gap-4, p-4 | Default element spacing, card padding |
| lg | 24px | gap-6, p-6 | Section padding, drop zone padding |
| xl | 32px | gap-8, p-8 | Page horizontal padding (mobile) |
| 2xl | 48px | gap-12, p-12 | Drop zone internal spacing, major section breaks |
| 3xl | 64px | gap-16, p-16 | Page vertical padding (desktop) |

Exceptions: Drop zone has a minimum height of 200px (not a spacing token -- a component dimension) to provide a large visual target per D-01.

## Typography

| Role | Size | Weight | Line Height | Tailwind Class |
|------|------|--------|-------------|----------------|
| Body | 14px | 400 (regular) | 1.5 | text-sm font-normal leading-normal |
| Label | 14px | 500 (medium) | 1.4 | text-sm font-medium |
| Heading | 20px | 600 (semibold) | 1.2 | text-xl font-semibold leading-tight |
| Display | 28px | 700 (bold) | 1.2 | text-2xl font-bold leading-tight |

**Rationale:** 14px body is compact enough for data tables (directory listing, error rows) while remaining legible on mobile. 20px heading provides clear hierarchy without overwhelming the functional UI. Display size (28px) is used only for the page title on the admin page.

**Print typography overrides (directory page only):**

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 10pt | 400 | 1.4 |
| Heading | 14pt | 600 | 1.2 |
| Display | 18pt | 700 | 1.1 |

## Color

Uses shadcn/ui CSS custom properties (HSL-based). Specific hex values are determined by the shadcn preset at init time. The contract defines roles, not absolute colors.

| Role | CSS Variable | Usage |
|------|-------------|-------|
| Dominant (60%) | `--background` | Page background, surfaces |
| Secondary (30%) | `--card`, `--muted` | Cards, drop zone background (idle), table header, directory groupings |
| Accent (10%) | `--primary` | Primary CTA buttons, import success indicator, active tab underline on directory page |
| Destructive | `--destructive` | "Replace All" button in the import mode dialog, error badges, validation error text |

**Accent reserved for (explicit list):**
1. "Import File" / file picker CTA button
2. Import success count badge (e.g., "87 flags imported")
3. "Print Directory" button on the directory page
4. Active/selected tab indicator on the directory page (Alphabetical vs By Row)

**Additional semantic colors:**

| Role | CSS Variable | Usage |
|------|-------------|-------|
| Warning | `--warning` (amber-500 / `#f59e0b`) | Warning badges for unrecognized columns (D-11), import warnings |
| Success | `--success` (emerald-600 / `#059669`) | Success state after import completes without errors |
| Muted foreground | `--muted-foreground` | Secondary text, row numbers in error list, empty state copy |

**Drop zone states (D-01):**

| State | Border | Background | Icon/Text |
|-------|--------|------------|-----------|
| Idle | 2px dashed `--border` | `--muted` at 50% opacity | `--muted-foreground` |
| Hover / Drag over | 2px dashed `--primary` | `--primary` at 5% opacity | `--primary` |
| Processing | 2px solid `--primary` | `--muted` at 50% opacity | Spinner in `--primary` |
| Success | 2px solid success | success at 5% opacity | Check icon in success |
| Error | 2px solid `--destructive` | destructive at 5% opacity | Alert icon in `--destructive` |

## Layout Contract

### Admin Page (/admin) -- D-07: Single page, import at top, summary below

```
+--------------------------------------------------+
|  [Page Title: Flag Data Import]                   |
|                                                   |
|  +----------------------------------------------+|
|  |                                              ||
|  |     [Upload Icon]                            ||
|  |     Drop your spreadsheet file here          ||
|  |     Drag and drop or click to browse         ||
|  |                                              ||
|  +----------------------------------------------+|
|  [Download sample template]                       |
|                                                   |
|  --- separator ---                                |
|                                                   |
|  [Import Summary: 87 flags, 3 errors]  [badges]  |
|                                                   |
|  [v] Import Errors (3)                            |
|      Row 5: Name is required                      |
|      Row 12: Invalid position "-1"                |
|      Row 23: Duplicate position (Row A, Pos 3)    |
|                                                   |
|  [Imported Flags - Table]                         |
|  | Name          | Row | Position |               |
|  |---------------|-----|----------|               |
|  | John Smith    | A   | 1        |               |
|  | Jane Doe      | A   | 2        |               |
|  | ...           | ... | ...      |               |
+--------------------------------------------------+
```

- Max content width: 768px (centered, `max-w-3xl mx-auto`)
- Page padding: 16px horizontal on mobile, 32px on desktop
- Drop zone: full width within content area, 200px min-height
- Summary and error list appear only after import
- Table shows all imported flags with sort-by-name default

### Directory Page (/directory) -- D-14, D-15

```
+--------------------------------------------------+
|  [Page Title: Flag Directory]     [Print Button]  |
|                                                   |
|  [Alphabetical] [By Row]          <- tabs         |
|                                                   |
|  -- Alphabetical View --                          |
|  Adams, John .............. Row A, Position 3     |
|  Baker, Mary .............. Row B, Position 12    |
|  Clark, Robert ............ Row A, Position 7     |
|  ...                                              |
|                                                   |
|  -- OR By Row View --                             |
|  Row A                                            |
|    Position 1: Smith, John                        |
|    Position 2: Doe, Jane                          |
|    Position 3: Adams, John                        |
|  Row B                                            |
|    Position 1: Williams, David                    |
|    ...                                            |
+--------------------------------------------------+
```

- Max content width: 768px (centered, `max-w-3xl mx-auto`)
- Print button hidden in `@media print`
- Tabs hidden in `@media print` (print both views, or active view only)
- Alphabetical view: two-column layout in print, single column on screen
- By-row view: grouped sections with `break-inside: avoid` for row groups

## Interaction States

### Drop Zone Interactions

| Trigger | State Change | Visual Feedback |
|---------|-------------|-----------------|
| Page load (no data) | Idle | Dashed border, muted bg, upload icon, instructional text |
| Page load (data exists) | Idle with summary | Drop zone idle + summary section visible below separator |
| Mouse enters drop zone | Hover | Border color transitions to primary, subtle bg tint |
| File dragged over zone | Drag active | Border becomes primary dashed, bg becomes primary at 5%, text changes to "Drop file to import" |
| File dragged away | Return to idle/hover | Revert border and bg |
| File dropped (new import) | Processing | Solid primary border, spinner replaces icon, "Importing flag data..." text |
| File dropped (data exists) | Dialog opens | Import mode dialog appears (D-13) before processing |
| Import succeeds (0 errors) | Success | Green border, check icon, success message with count |
| Import succeeds (with errors) | Partial success | Revert to idle, show summary with success badge + error badge, expand error list |
| Import fails completely | Error | Red border, alert icon, error message |
| Click on drop zone | File picker opens | Native file dialog |

### Directory Interactions

| Trigger | State Change | Visual Feedback |
|---------|-------------|-----------------|
| Page load | Alphabetical tab active | Tab underline on "Alphabetical", list renders sorted A-Z |
| Click "By Row" tab | Tab switch | Underline moves to "By Row", content switches to grouped view |
| Click "Print Directory" | Browser print dialog | Browser native print dialog opens, print CSS applied |
| No data imported | Empty state | Empty state message with link to admin/import page |

## Responsive Behavior

| Breakpoint | Admin Page | Directory Page |
|------------|-----------|----------------|
| Mobile (< 640px) | Full-width drop zone, p-4 horizontal padding, stacked layout | Single column list, tabs stack if needed, p-4 padding |
| Tablet (640-1024px) | max-w-3xl centered, p-6 padding | max-w-3xl centered, p-6 padding |
| Desktop (> 1024px) | max-w-3xl centered, p-8 padding | max-w-3xl centered, p-8 padding |

All pages use a single-column centered layout. No sidebar or multi-column screen layouts for Phase 1.

## Accessibility Requirements

| Requirement | Implementation |
|-------------|---------------|
| Drop zone keyboard accessible | react-dropzone provides keyboard support (Enter/Space to open file dialog) |
| Dialog focus trap | shadcn Dialog (Radix) handles focus trap automatically |
| Error announcements | Use `role="alert"` on import error/success messages for screen reader announcement |
| Color contrast | shadcn defaults meet WCAG 2.1 AA contrast ratios |
| Tab navigation | All interactive elements (buttons, tabs, links) reachable via Tab key |
| Print a11y | Semantic HTML tables for directory listings; `thead` repeats on print pages |
