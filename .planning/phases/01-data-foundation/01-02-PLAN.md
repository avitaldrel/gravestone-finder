    - validateRows() with valid CSV data returns { valid: FlagInsert[], errors: [], warnings: [] }
    - Each valid entry has { name: string, row_label: "A"|"B"|..., position: number }
    - Row with empty name produces error: { row: N, issues: ["Name is required"] }
    - Row with empty row produces error: { row: N, issues: ["Row is required"] }
    - Row with empty position produces error: { row: N, issues: ["Position must be a positive number"] }
    - Row with non-numeric position produces error for position
    - Row with position 0 or negative produces error
    - Duplicate (row_label, position) pairs produce error on the second occurrence: "Duplicate position: Row {X}, Position {Y} already assigned"
    - Completely empty rows (all cells blank/undefined) are silently skipped -- no error
    - Numeric row identifiers are normalized: row "1" -> row_label "A"
    - Row numbers in errors are 1-indexed (human-readable)
    - Extra columns produce a warning: "Unrecognized columns ignored: {col1}, {col2}" (D-11)

  <implementation>
    Implement in this order (each module follows RED -> GREEN -> REFACTOR):

    **Module 1: normalize-row.ts**
    - Export `normalizeRowLabel(input: string | number): string`
    - Trim and uppercase string input
    - If /^[A-Z]+$/ matches, return as-is (already a letter)
    - If parseable as positive integer, convert using number-to-letters algorithm (1=A, 2=B, ..., 26=Z, 27=AA)
    - Otherwise throw Error("Invalid row identifier: \"{input}\"")

    **Module 2: detect-headers.ts**
    - Export `detectHeaders(firstRow: unknown[]): { hasHeaders: boolean; columnMap: Record<string, number>; unrecognized: string[] }`
    - Normalize cells to lowercase strings
    - Count matches against ["name", "row", "position"]
    - If >= 2 matches: hasHeaders=true, build columnMap from actual positions, collect unrecognized column names (D-11)
    - If < 2 matches: hasHeaders=false, columnMap defaults to { name: 0, row: 1, position: 2 }, unrecognized=[]

    **Module 3: parse-spreadsheet.ts**
    - Export `parseSpreadsheet(file: File): Promise<unknown[][]>`
    - Read file as ArrayBuffer using file.arrayBuffer()
    - Call SheetJS `read(new Uint8Array(buffer), { type: "array" })`
    - Extract first sheet: `workbook.Sheets[workbook.SheetNames[0]]`
    - Convert to array of arrays: `utils.sheet_to_json(sheet, { header: 1 })`
    - Wrap in try/catch, rethrow as `Error("Could not read this file. Please check that it is a valid spreadsheet and try again.")`
    - Use named imports: `import { read, utils } from "xlsx"` (NOT default import -- SheetJS v0.20.x uses named exports)

    **Module 4: validate-flags.ts**
    - Export `validateRows(rawRows: unknown[][]): ImportResult`
    - Call detectHeaders(rawRows[0]) to get columnMap, hasHeaders, unrecognized
    - Set startRow = hasHeaders ? 1 : 0
    - If unrecognized.length > 0, add warning: "Unrecognized columns ignored: {joined names}. Only Name, Row, and Position columns are imported."
    - Create Zod schema: z.object({ name: z.string().min(1, "Name is required").trim(), row: z.string().min(1, "Row is required").trim(), position: z.coerce.number().int().positive("Position must be a positive number") })
    - Track seen positions with Set<string> for duplicate detection
    - For each row starting at startRow:
      - Skip if all cells empty/undefined/blank
      - Extract name, row, position using columnMap indices
      - safeParse with Zod schema
      - If invalid: add to errors array with 1-indexed row number
      - If valid: normalizeRowLabel on row field. Check for duplicate `${row_label}-${position}`. If duplicate, add error. Otherwise add to valid array and seen set.
    - Return { valid, errors, warnings }
  </implementation>
